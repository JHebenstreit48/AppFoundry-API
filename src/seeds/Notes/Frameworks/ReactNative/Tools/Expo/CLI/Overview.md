## Introduction to Expo CLI

<span class="emphasis">Expo CLI</span> is a free, open-source toolchain built on top of React Native,
providing a streamlined and managed workflow for building cross-platform apps.

It removes the complexity of manual native setup and lets developers focus purely on JavaScript and UI logic.

---

### Why Use Expo CLI?

Using <span class="emphasis">Expo CLI</span> makes React Native development faster and simpler — perfect for
teams or individuals who want to avoid manual Android/iOS configuration.

**Key Advantages:**

- No Xcode or Android Studio required for basic builds.
- Access to prebuilt device APIs via the Expo SDK.
- Fast refresh, easy debugging, and over-the-air (OTA) updates.
- Seamless integration with the <span class="emphasis">Expo Go</span> app for testing.

For setup and workflow steps, see  
<a href="/react-native/tools/expo/cli/workflow">Expo — CLI Workflow</a>.

For ejecting into a full native project, see  
<a href="/react-native/tools/expo/cli/eject">Ejecting from Expo</a>.

### Summary

Expo CLI offers a managed approach that accelerates development while maintaining flexibility to switch to a
bare React Native workflow later.
